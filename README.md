# employee-feature-engineering

# 📊 Employee Feature Engineering & Data Preprocessing Project

## 📌 Project Overview
This project demonstrates complete end-to-end feature engineering and data preprocessing on a messy employee dataset.

The goal is to transform raw, inconsistent data into a clean, model-ready dataset suitable for machine learning.

---

## 🚀 Topics Covered

### 🔹 Data Cleaning
- Handling duplicates
- Fixing inconsistent categorical values
- Cleaning mixed-format numeric columns
- Handling invalid entries

### 🔹 Missing Value Treatment
- Mean imputation
- Median imputation
- Mode imputation
- Group-based imputation
- Forward fill & backward fill

### 🔹 Categorical Encoding
- Label Encoding
- One-Hot Encoding
- Ordinal Encoding
- Target / Mean Encoding
- Frequency Encoding

### 🔹 Outlier Detection
- IQR Method
- Z-Score Method
- Winsorization (Capping)

### 🔹 Feature Engineering
- Date-based features (Tenure, Join Year, Quarter)
- Salary per experience
- Experience-age ratio
- Seniority flag
- Performance-to-salary ratio

### 🔹 Binning / Bucketing
- Equal-width binning
- Quantile-based binning
- Custom grade creation
